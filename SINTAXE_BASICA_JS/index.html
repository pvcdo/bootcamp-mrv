O que é o JS

Linguagem de alto nível (parece com a linguagem humana) para dispositivos e páginas web. Uma das mais usadas em desenvolvimento.

Outra pessoa que deu a dica de carregar os scripts ao final do body (isso porque o js pode querer interagir com algum elemento ainda não renderizado se for declarado antes).
