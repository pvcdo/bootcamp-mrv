Requisitos do projeto

O professor indicou fazer o download do SDK do .NET no site da Microsofte também o VSCode. Irei fazer essa instalação, mesmo tendo o visual studio e instalando as "cargas de trabalho" como indicado pela nossa querida colega Giovanna Ramos.

  1. criar pasta do raiz projeto
  2. abrir o cmd nesse diretório
  3. comando dotnet new sln --name Colecoes
    Esse comando cria uma "solution" com o nome Colecoes. Essa solution é um agrupamento para o projeto.
  4. deu o comando dotnet new console --name Colecoes para criar a pasta do projeto com os arquivos necessários.
  5. dotnet sln add Colecoes/Colecoes.csproj
    Esse comando adiciona na solution o arquivo csproj especificado.
  6. abriu a pasta do raiz do projeto no vscode
  7. no vscode ele instalou as extensões
    • c#
    • c# extensions
    • vscode-icons
  8. dentro da pasta Colecoes ele criou uma outra chamada Models

Classe Array

Elemento existe?

  • Array.Exists(array, elemento_array => elemento_array == valor_procurado)

    Verifica se o valor de cada elemento (elemento_array) do array passado (array) é igual ao valor_procurado.

    Também podemos fazer outro tipo de comparação não apenas igualdade. Podemos por exemplo verificar se há algum valor maior que o valor procurado trocando o == por >

Comparando todos os elementos do array

  • Array.TrueForAll(array, elemento_array => elemento_array > valor_procurado)

    Vê se a comparação estabelecida é verdadeira com todos os elementos do array. No exemplo estamos verificando se todos os valores do array são maiores que o valor procurado, caso algum não seja o retorno é "False"

Encontrando um elemento no array

  • Array.Find(array, el => el == valor_procurado)

    Encontra o valor e retorna sua primeira ocorrência.

    Caso não ache o valor procurado, haverá o retorno do valor padrão do tipo de array (no caso de inteiros o valor padrão é 0)

  • Array.FindAll

    Retorna um array "com a condição que eu busquei"

  • Array.FindIndex(array, el => el == valor_procurado)

    retorna a posição da primeira ocorrência do valor procurado

  • Array.FindLast

    Retorna o valor que eu estou buscando, porém sua última ocorrência

  • Array.FindLastIndex

    Retorna o index da última ocorrência que eu estou procurando

  • Array.IndexOf(array, valor_procurado)

    É a mesma coisa que fazer o Array.FindIndex com a compração de igualdade.

    Caso não encontre o valor procurado o retorno é de -1.

Redimensionando o array

  • Array.Resize(ref array, novoTamanho)

    O array retornado tem os mesmos itens iniciais, as posições adicionadas tem itens de valor igual a 0 (no caso de arrays de inteiros).

Convertendo um array de um para outro tipo

  • Array.ConvertAll(array, el => el.ToString())

    O exemplo é a conversão de um array para o tipo String.
