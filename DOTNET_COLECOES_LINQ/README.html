Requisitos do projeto

O professor indicou fazer o download do SDK do .NET no site da Microsofte também o VSCode. Irei fazer essa instalação, mesmo tendo o visual studio e instalando as "cargas de trabalho" como indicado pela nossa querida colega Giovanna Ramos.

  1. criar pasta do raiz projeto
  2. abrir o cmd nesse diretório
  3. comando dotnet new sln --name Colecoes
    Esse comando cria uma "solution" com o nome Colecoes. Essa solution é um agrupamento para o projeto.
  4. deu o comando dotnet new console --name Colecoes para criar a pasta do projeto com os arquivos necessários.
  5. dotnet sln add Colecoes/Colecoes.csproj
    Esse comando adiciona na solution o arquivo csproj especificado.
  6. abriu a pasta do raiz do projeto no vscode
  7. no vscode ele instalou as extensões
    • c#
    • c# extensions
    • vscode-icons
  8. dentro da pasta Colecoes ele criou uma outra chamada Helper

Ordenação de arrays

  1. Em Helper ele criou as classes OperacoesArray
  2. criou um método public void que recebe como parâmetro um array de inteiros. Esse método percorre o array e imprime cada um dos seus itens
  3. criou um metodo public void que também recebe um array de inteiros porém agora por referência e escreveu um código bubble sort em c#

    Receber por referência indica que o array recebido será alterado, ele quem será afetado com as intruções do método, não haverá o retorno de um novo array.

Classe Array

  1. o professor criou na classe OperacoesArray um método que ordena um array a partir da classe Array pelo seu método Array.Sort(ref array)
  2. também um método que copia um array para outro pelo método Array.Copy(ref array_0, ref array_destino, tamanho_copia)
