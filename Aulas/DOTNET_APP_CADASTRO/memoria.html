Revisão

Classes abstratas

São classes que podem conter métodos abstratos. Essas classes não são instanciadas, mas só servem como "base" para outras classes, porém, as classes que herdam seus métodos e atributos devem implementá-los. As subclasses herdam as classes (1 classe-filha => 1 classe-mãe).

Interfaces

Semelhante a classes abstratas, mas não possui atributos. Não pode definir como os métodos devem ser implementados. Basicamente é uma lista de métodos que devem ser implementados. As classes implementam interfaces (1 classe-filha => n interfaces).
