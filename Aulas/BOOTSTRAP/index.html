Aprenda sobre o framework Bootstrap

O professor baixou o bootstrap na máquina... eu geralmente utilizo o link cdn. Nessa disciplina o professor vai explorar apenas o código CSS.

Uma dica muito importante foi que é possível consultar os códigos fontes das páginas de exemplo do site do bootstrap e ir montando as nossas páginas com base nesses exemplos.

Criou a página index.html na pasta do projeto, chamando o css do bootstrap e chamando um css próprio.

Ele colocou uma navbar do bootstrap no index.

Ele criou no css próprio uma classe .navbar, sobrepondo o dado de background-color que vem do bootstrap.

Depois ele colocou uma section com id #quem-somos e dentro dela uma div com as classes bootstrap (cb) .container-fluid e .text-center e classe própria (cp) .quem-somos, para alterar o background-color e o padding. A classe text-center vai justificar ao centro tudo que estiver dentro desta div.

Dentro da div ele colocou um h3 com o inner "Quem somos" e abaixo dele uma img com width=400 e height=400 e puxou a cb .rounded-circle para essa imagem, para fazer um efeito de arredondamento nesta imagem. Ele mudou a fonte do body e as cores.

Criou uma section#parceiros, dentro dela uma div.container-fluid e dentro desta uma div.container (neste caso começou a fazer a criação das doze colunas), dentro desta uma div.row e a partir daí criou 3 div.col-md-4 (sendo que é possível associar mais tipos de tamanhos de dispositivos; por exemplo col-md-4 col-sm-2 col-lg-3 (sendo assim em dispositivos de tamanho médio será uma área de 4 colunas, em pequenos de 2 e grandes de 3)). Dentro de cada área ele criou uma imagem com width=100%
