O que é o DOM

É um padrão de como manipular elementos HTML de uma página.

BOM

Browser Object Model é o objeto que representa a janela do browser. É acessado nos browsers pela palavra "window". O BOM é o pai do DOM.

Métodos

Estrutura:

  • tag => ex: <p> / <body> / <html>
  • id => identificador único
  • class => identificador múltiplo

Nós podemos a partir do DOM (document) procurar os elementos da página html por sua tag, id ou class, sendo que o id é o método que vai retornar apenas um elemento, os outros retornam um array.

Outro seletor é o querySelectorAll, no qual passamos tag, classe ou id e volta um array com esses elementos.

É sempre interessante selecionar os elementos pelo id, pensando na escalabilidade que o site pode ter.

  • document.createElement(element) = criar um elemento html na página
  • document.removeChild(element) = remove filhos da página
  • document.appendChild(element) = adiciona um elemento na página
  • document.replaceChild(element) = substitui um elemento na página

Trabalhando com estilos

Element.classList => representa a lista de classes

  • .add = adiciona uma classe no elemento
  • .remove = remove
  • .toggle = adiciona a classe se ela não existe e remove se ela existe.

Para colocar um css direto pelo javascript nós usamos Element.style.[propriedade]

Eventos

São ações que o usuário faz na página. Os mais comuns são:

  • mouseouver, mouseout
  • click, dbclick (duplo clique)
  • change (quando um campo muda), load (quando o elemento acaba de carregar)

Para acioná-lo nós damos Element.addEventListener('evento',função). Para colocá-lo diretamente pelo html os termos vem com o termo "on" na frente do evento (ex.: click (js) => onclick (html)).
