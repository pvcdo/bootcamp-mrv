Instalação do Visual Studio no windows

Download Visual Studio

Instalar a versão comunidade. Depois de instalar o visual studio devemos instalar as "cargas de trabalho" para "Desenvolvimento para desktop com .NET".

Gerando código e debugging

Criamos um projeto novo na área de trabalho. Baseamos o nosso porjeto como um "Aplicativo do Console". Depois abre-se a página para escolhemos o nome do projeto, que inicialmente coincide com o "Nome da Solução". Essa solução é onde ficarão todos os projetos integrados. Damos os next e é criado um projeto inicial já com alguns códigos.

A depuração é feita quando colocamos um breakpoint no código e ele é executado. Bem parecido com o VBA, como já imaginávamos. Para dar o bradkpoint nós podemos clicar na coluna de linhas ou dando f9 na linha que queremos. Depois de marcada a linha de depuração nós executamos o código.

Integração com o GitHub

Podemos clonar um repositório a partir da página inicial do visual studio clicando no botão "Clonar um repositório".

Outra forma é criar um novo repositório para um projeto já existente. Para isso vamos no menu Git e depois em Criar um repositório Git, e passar o que é requisitado.

Testes

É possível criar testes unitários dentro do visual studio selecionando o nome da classe que queremos testar, depois clicando com o botão direito e criar teste de unidade, após isso é criado um documento no qual temos que passar os parâmetros para que o programa interaja e veja se o que é esperado é o que realmente acontece.
