Map, filter e reduce

São métodos importantes de arrays.

Map

Ela cria um novo array novo que é a imagem de uma função submetida ao array principal. O array principal não é modificado

    array.map(function(item,[index,[array]]){
        //instrução que faz alguma operação no item
    }, [thisArg])
Importante notar que com arrow function o método pode dar errado.

Map vs forEach

O forEach não retornar um novo array, ele apenas executa a função em cada um dos itens do array principal. Caso queiramos retornar um array temos que antes ter um array secundário e fazer um push dos resultados do forEach.

Filter

Também cria um novo array, mas apenas com os itens que atenderam a classificação do filter.

    array.filter(function(item,[index,[array]]){
        //instrução que verifica um true ou false em cada item
    }, [thisArg])

Reduce

Não retorna um novo array. Ele faz uma função que irá interagir com todos os elementos do array e trazer um único valor.

    array.reduce(function(acumulator,currentValue,[index,[array]]){

    },[initialValue])

Se o initialValue não for passado ele assumirá na primeira iteração o primeiro valor do array e o currentValue será o segundo, seguindo as iterações normalmente; caso o initialValue seja passado, o valor do currentValue será o primeiro valor do array.
