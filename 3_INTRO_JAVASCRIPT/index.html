JavaScript em uma página da web

Arquitetura é a forma como nossos arquivos e pastas estão organizados.

Nós separamos o index.html dos demais arquivos, colocando eles em outras pastas. Geralmente arquivos js, css, img, estão na pasta assets.

Caso o script (arquivo js) seja muito grande o ideal é que ele seja chamado antes da tag de encerramento do html.

Para importar um script nós damos o comando <script scr='path'></script>
